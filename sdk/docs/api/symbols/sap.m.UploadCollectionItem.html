new sap.m.UploadCollectionItem(sId?, mSettings?)
Constructor for a new UploadCollectionItem.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • contributor : string
    • documentId : string
    • fileName : string
    • fileSize : float
    • mimeType : string
    • thumbnailUrl : string
    • uploadedDate : string
    • url : string
    • enableEdit : boolean (default: true)
    • enableDelete : boolean (default: true)
    • visibleEdit : boolean (default: true)
    • visibleDelete : boolean (default: true)

In addition, all settings applicable to the base type sap.ui.core.Element can be used as well.

Parameters:
{string} sId? id for the new control, generated automatically if no id is given
{object} mSettings? initial settings for the new control
