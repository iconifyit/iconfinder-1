Tutorials

Learn how to develop applications with OpenUI5 by following the tutorials.

In this section:
  • Create Your First OpenUI5 Application
    You can include a bootstrap to the OpenUI5 JavaScript libraries to use OpenUI5 in an HTML page without having set up the OpenUI5 application development tools in Eclipse.
  • Create Your First Mobile OpenUI5 Application
    Whether you develop a OpenUI5 application for desktop or for mobile, the concepts are very similar. OpenUI5 provides an additional control library called sap.m, which is optimized for mobile devices, but also runs fine in desktop browsers.
  • Application Best Practices
    This guide tells you what you need to know to build non-trivial enterprise scale applications (apps) with OpenUI5.
