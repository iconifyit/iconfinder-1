Browser and Platform Matrixes

The browser and platform information in the following sections provides you the information you need to decide, which browsers and platforms suit best to your use case.

As OpenUI5 bases on CSS3, HTML5, and the new JavaScript API, only browsers with HTML5 capabilities are supported.

Depending on the platform your OpenUI5 applications runs on, different browsers in different versions are supported. If you know the platform and the browser used by your users, you can decide on the libraries you use for your application.

In this section:
  • sap.ui.core, sap.ui.layout, sap.ui.unified
    Basic libraries, supporting all platforms or browsers that are supported by any of the other libraries
  • sap.ui.commons, sap.ui.ux3, sap.ui.table
    Browser support for sap.ui.commons, sap.ui.ux3, sap.ui.table libraries on Microsoft Windows and Mac OS platforms.
  • sap.m
    Browser support for sap.m library on iOS, Android, BlackBerry, MacOS, and Windows platforms
  • sap.suite.ui.commons
    Browser support for sap.suite.ui.commons library on iOS, Android, BlackBerry, MacOS, and Windows platforms
