Installing OpenUI5

The installation guide helps administrators with the required steps to install, configure, and maintain the UI development toolkit for HTML5 (OpenUI5).

Note

You can install OpenUI5 on different platforms. The license and maintenance conditions of the respective platforms apply also for OpenUI5. If you use OpenUI5 tools on SAP HANA Cloud Platform, for example, the licence and maintenance conditions of the SAP HANA Cloud Platform apply.
